BalloonsFighter



開発環境
Eclipse
使用言語
Java
制作期間
1ヶ月
ジャンル
アクションゲーム
操作方法
↑キー　上昇,　←→キー　左右移動, Pキー ゲームスタート,　Rキー　リトライ
作品紹介
横スクロールの2Dアクションゲーム
重力落下するプレイヤーを上手く操作して障害物や敵を避けつつゴールを目指すゲームとなります。
クリア条件はゴール座標へ到達することでゲームクリアとなります。
参考書のサンプルコードを基盤に自身で考えた作品に改良したものとなりますが、mapファイルで生成したマップを横スクロールさせる所で難航しました。
